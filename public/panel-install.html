วิธีติดตั้ง Arbi Gen5

ตั้งค่าครั้งเดียว — ลูกค้าไม่ต้องใส่ URL หรือรหัสผ่าน

บัญชีต้องเป็นโหมด Hedging — ถ้าเป็น Netting EA จะไม่ทำงาน

ขั้นตอนติดตั้ง

  1. ดาวน์โหลดไฟล์
    ได้ Arbi_Gen5.ex5 จาก หน้าดาวน์โหลด
  2. คัดลอกไฟล์เข้าโฟลเดอร์ Experts
    เปิด MT5 → เมนู File → Open Data Folder → MQL5 → Experts
    วางไฟล์ Arbi_Gen5.ex5 ลงในโฟลเดอร์ Experts (หรือ Experts ย่อย)
  3. รีเฟรช Navigator
    คลิกขวาที่ Navigator → Refresh — จะเห็น Arbi_Gen5 ภายใต้ Expert Advisors
  4. เปิดการตั้งค่า Expert Advisors
    Tools → Options → Expert Advisors แล้วเปิด:
    • Allow algorithmic trading
    • Allow DLL imports (จำเป็น — ใช้เชื่อมต่อเซิร์ฟเวอร์ license)
    ไม่จำเป็นต้องใส่ WebRequest URL ถ้าใช้ DLL ได้
  5. ลาก EA ลงชาร์ต
    ลาก Arbi_Gen5 ไปที่ชาร์ตใดก็ได้ (แนะนำ symbol หลักของบัญชี)
    ในหน้าต่าง Inputs ไม่ต้องแก้ URL หรือ API key — ฝังมาจากผู้พัฒนาแล้ว
  6. อนุญาตการเทรดอัตโนมัติบนชาร์ต
    กดปุ่ม AutoTrading ให้เป็นสีเขียว และอนุญาตเมื่อ EA ถามสิทธิ์
  7. ตรวจสอบในแท็บ Experts
    ควรเห็นข้อความประมาณ EaPlatform: license approved หรือ trial until …
    EA จะตรวจ license วันละครั้ง — ถ้าถูกปิดจากผู้ดูแลจะถอนตัวออกจากชาร์ตอัตโนมัติ

แก้ปัญหาเบื้องต้น

  • INIT_FAILED / DLL — เปิด Allow DLL imports แล้ว restart MT5
  • ไม่ใช่ Hedging — เปลี่ยนบัญชีหรือโบรกเกอร์ที่รองรับ hedging
  • Strategy Tester — Arbi_Gen5 ใช้ได้เฉพาะ live เท่านั้น
  • หมด trial / ถูกปิด — ติดต่อผู้ดูแลเพื่อเปิดสิทธิ์บนระบบ
